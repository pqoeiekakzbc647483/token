## 🚀 快速部署
### 开发工具推荐
- Remix IDE（网页版，无需配置）
- Hardhat / Truffle

### 部署步骤
1. 打开合约文件
2. 确认编译器版本（^0.8.0）
3. 连接钱包，选择网络部署
4. 部署完成后调用合约方法
